# DSS SUITE

**DSS SUITE** es una colección integral de herramientas y optimizaciones para WordPress, diseñada para estandarizar la arquitectura y mejorar la experiencia administrativa en los proyectos de **DSS NETWORK**.

## 🚀 Módulos Incluidos

La suite se compone de varios módulos especializados que se pueden gestionar de forma centralizada:

### 1. 🏗️ White Label (DSS Themes)

Personaliza la interfaz de WordPress para ofrecer una experiencia de marca blanca coherente.

- **Widgets de Dashboard personalizados**: Añade herramientas útiles para el cliente final.
- **Limpieza de Interfaz**: Elimina o renombra elementos innecesarios del menú administrativo.
- **Branding**: Personaliza la apariencia visual del panel de administración.

### 2. 📊 DSS Dashboard

Un rediseño total de la interfaz administrativa de WordPress orientado a la usabilidad.

- **Panel de Control Centralizado**: Acceso rápido a las configuraciones de la suite.
- **Estética Moderna**: Mejora visual de menús, colores y tipografía.
- **Configuraciones de Tema**: Gestión de navegación, colores y pie de página administrativo.

### 3. 📑 CPT Sorter

Añade funcionalidad de ordenamiento intuitivo para tipos de contenido personalizados.

- **Drag & Drop**: Ordena entradas de portfolio (`cpt_portfolio`) arrastrando y soltando en el panel de administración.
- **Auto-Sync**: El orden establecido en el admin se aplica automáticamente en el frontend de la web.
- **Interfaz Nativa**: Integrado perfectamente con las tablas estándar de WordPress.

### 4. 🏷️ SEO Manager (Tag Manager Integration)

Simplifica la implementación de herramientas de seguimiento y análisis.

- **Google Tag Manager**: Integración sencilla del contenedor de GTM en todas las páginas.
- **Optimización de Carga**: Inserción eficiente de scripts para no penalizar el rendimiento.

## 🛠️ Instalación

1. Sube la carpeta `dss-suite` al directorio `/wp-content/plugins/`.
2. Activa el plugin desde el menú **Plugins** en el escritorio de WordPress.
3. Accede a **DSS Dashboard** para configurar los módulos activos.

## 📝 Requisitos

- WordPress 5.0 o superior.
- PHP 7.4 o superior.
